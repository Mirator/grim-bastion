# Asset Manifest

This project currently uses procedural/generated assets only.

## Visual Assets
- Tower, enemy, trap, hero, and environment meshes are generated at runtime using Three.js primitive geometry.
- No external model packs are bundled in the repository.

## Audio Assets
- Sound effects are generated at runtime as procedural WAV buffers and played through Howler.
- No external audio packs are bundled in the repository.

## License Policy
- External assets, if added later, must be CC0/Public Domain.
- Add any imported asset source + license details to `docs/CREDITS.md`.
